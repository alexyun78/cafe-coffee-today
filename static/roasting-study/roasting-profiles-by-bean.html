②모든 로스팅의 공통 뼈대

생두가 뭐든 항상 똑같이 지키는 원칙이에요. 이게 흔들리면 생두별 조정은 의미가 없어요.

  • 4단계 흐름: 건조 → 마이야르(갈변) → 1차 크랙 → 디벨롭 → 드롭. 순서는 안 바뀌어요.
  • RoR은 매끄럽게 우하향: 계속 떨어지되 멈추거나(크래시) 다시 튀지(플릭) 않게. 곡선의 '모양'이 숫자보다 중요.
  • 1차 크랙 직전 화력 감소: 발열 폭주 대비. 특히 할로겐(복사)·드럼히터(전도)부터 먼저 낮춰요.
  • 색만 보고 판단 금지: 소리·시간·RoR·향을 종합. (내추럴·디카페인은 특히 색이 거짓말을 해요)
  • 교반으로 균일성 확보: 한쪽만 타는 걸 막아요. 밀도 낮고 불균일한 콩일수록 더 높게.
  • 기준점 고정: 투입 170 · FC 174~177 · 드롭 180+. 여기서 ±로 조정.
  • 일지 + 리플리케이션: 좋은 배치를 기록하고 복제. (Boost 데이터의 70%가 리플리케이션)[Boost]
열원 역할 복습 (→ 모듈 07) 열풍=대류(균일·깔끔) · 할로겐=복사(표면·크랙 추진, 과하면 스코칭/팁핑) · 드럼히터=전도(바디, 과하면 스코칭) · 교반(균일성). 생두별 조정은 결국 "이 네 축의 배분을 어떻게 바꾸느냐"예요.

④가공별 분기 — 워시드 · 내추럴 · 무산소

⑤산지별 분기 — 브라질 · 에티오피아 · 케냐 · 인도네시아

산지는 주로 고도→밀도와 산미 성격으로 갈려요. 밀도가 높으면 열이 더 들고, 낮으면 겉이 타기 쉬워요. (→ 모듈 02)

⑦공통 vs 분기 — 한 장 정리

무엇을 항상 똑같이 가져가고, 무엇을 생두별로 손보는지 한눈에.

요소공통 / 분기내용
4단계 흐름공통건조→마이야르→1차크랙→디벨롭→드롭 (순서 불변)
RoR 관리공통매끄러운 우하향, 크래시·플릭 방지
크랙 전 화력↓공통할로겐·드럼히터부터 단계적으로
판단 기준공통색만 X → 소리·시간·RoR·향 종합
1차 크랙 온도공통거의 모든 생두 Internal 174~177℃
일지·리플리케이션공통골든 배치 기록·복제
투입(차징) 온도분기디카페인 160~165 · 저밀도/내추럴 165~168 · 워시드 170 · 고밀도(케냐) 170~172
열원 배분분기저밀도·내추럴=할로겐↓·교반↑ / 고밀도=에너지↑ / 디카페인=전체↓
디벨롭 길이(DTR)분기향 보존(에티오피아)=짧게 12~18% / 단맛·바디(브라질·인니)=길게 18~25%
로스트 레벨·드롭분기라이트 ~180 / 미디엄 ~182 / 미디엄다크 ~185~187 / 다크 188℃+
주의 결함분기저밀도=스코칭·팁핑 / 고밀도=언더·베이킹 / 디카페인=스코칭·과볶음
한 문장 요약 뼈대(흐름·RoR·크랙 기준점·판단법)는 항상 같고, 투입 온도·열원 배분·디벨롭 길이·드롭 레벨 네 가지를 생두 성격(밀도·가공·목표 향미)에 맞춰 손봐요.

⑧KCRC 챔피언 프로파일 분석 Boost 실측
