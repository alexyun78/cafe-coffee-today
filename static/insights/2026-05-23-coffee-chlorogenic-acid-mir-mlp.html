커피 로스팅 중 클로로젠산 실시간 정량화: 미드 인프라레드 분광법과 다층 퍼셉트론의 융합

MIR 분광법과 다층 퍼셉트론(MLP) 회귀로 140~220°C 로스팅 범위에서 CGA 농도를 HPLC-DAD 수준으로 비파괴 예측

CHEMIE INGENIEUR TECHNIK 2026-03 DOI 10.1002/CITE.70049 화학/분석화학 · 장비/공학
MIR 분광법과 다층 퍼셉트론(MLP) 회귀로 140~220°C 로스팅 범위에서 CGA 농도를 HPLC-DAD 수준으로 비파괴 예측

🤔 무슨 연구야?

클로로젠산(CGA)은 커피 생두 건물량의 4~10%를 차지하는 주요 폴리페놀로, 항산화 활성·쓴맛·산미에 관여하며 로스팅 중 열분해된다. 현재 표준 정량법인 HPLC는 파괴적·시간 소모적이어서 실시간 공정 모니터링에 부적합하다. 만하임 응용과학대학교 CeMOS 연구팀은 MIR 분광법과 MLP 회귀 모델을 결합한 비파괴 CGA 정량 프레임워크를 개발하여, 140~220°C 로스팅 구간 44개 샘플 및 미로스팅 대조군에 적용했다. MLP 알고리즘은 스펙트럼 전처리와 CGA 특이적 파수 영역 자동 선택을 수행했으며, HPLC-DAD 대비 높은 예측 정확도와 일반화 능력을 달성했다. MIR은 NIR 대비 분자 진동 민감도가 높아 CGA 이성질체 구별에 더 유리하며, 이 연구는 분광-머신러닝 결합 모델의 커피 품질 모니터링 패러다임을 제시한다.

🎯 핵심 발견

  • MIR 분광법 + MLP 회귀로 140~220°C 로스팅 구간 45개 샘플에서 CGA 농도 예측 — HPLC-DAD 대비 높은 정확도 및 우수한 일반화 능력 확인
  • MLP 알고리즘이 스펙트럼 전처리와 CGA 특이적 파수 영역 자동 선택 수행 — 분석가 의존적 설정 최소화
  • 비파괴·비용매·단시간 MIR 기반 CGA 정량 프레임워크의 커피 로스팅 공정 적용 가능성 최초 실증
  • 와전류 로스터 환경에서 수집된 실제 공정 조건 데이터 사용
  • 동일 연구팀 Raman 분광법 선행 연구 대비 MIR의 CGA 분자 진동 민감도 우위 적용

💼 누구한테 의미가 있을까?

대상의미
농가 · 가공장로스팅 공정 분석 도구 연구로 재배·수확후 처리와 직접 관련 없음. 향후 MIR CGA 모니터링 상용화 시 생두 품질(CGA 초기 함량)과 로스팅 후 잔존량 관계의 정량적 추적 인프라 구축에 간접 기여 가능.
로스터 · 큐그레이더CGA 함량의 실시간 추적으로 원하는 CGA 잔존 범위에서 정확히 로스팅을 중단·전환하는 정밀 공정 제어 가능. 라이트 로스트에서 항산화 특성 극대화, 미디엄 로스트에서 배치 간 산미 일관성 확보 시나리오 구현 가능.
R&DMIR+MLP 프레임워크를 카페인·트리고넬린·멜라노이딘 등 다른 로스팅 지표 물질로 확장 가능. 드럼·유동층 로스터 등 상업 장비로 모델 전이 검증, 인라인 MIR 탐침 결합을 통한 실시간 자동화 연구가 다음 단계.

CGA (Chlorogenic Acid)
클로로젠산 — 커피 생두 주요 폴리페놀(건물 기준 4~10%). 항산화·항염 활성. 로스팅 중 열분해
MIR (Mid-Infrared Spectroscopy)
미드 인프라레드 분광법 — 4,000~400 cm⁻¹ 파장대로 분자 진동 직접 측정. NIR 대비 구조 정보 풍부
MLP (Multilayer Perceptron)
다층 퍼셉트론 — 완전 연결 인공신경망. 비선형 회귀에 강점
HPLC-DAD
고성능 액체 크로마토그래피-다이오드 어레이 검출기 — CGA 정량 참조 표준 방법
Eddy Current Roaster
와전류 로스터 — 전자기 유도로 금속 용기를 직접 가열. 균일·빠른 가열
Wavenumber Region Selection
파수 영역 선택 — CGA 특성 흡수 파수 구간을 알고리즘으로 자동 선별
